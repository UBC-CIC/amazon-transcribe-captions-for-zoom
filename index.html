Real-time Audio Transcription Close Captioning for Zoom

Using the Amazon Transcribe WebSocket API

Instructions:
  1. Launch a new Zoom meeting.
  2. In the meeting room task bar click on the 'Closed Caption' icon and click 'Copy the API token' to copy the API token to clipboard.
  3. Paste the API token below in the 'ZOOM CC API Token' field.
  4. Select a language in the 'Language' field if needed.
  5. Click the 'Start' button when you are ready to start recording audio to be transcribed. This will prompt a browser response to grant access to your microphone, click 'Allow'.

Muting your microphone in Zoom will not mute your microphone in this tool. You will still generate captions that will be sent to the Zoom meeting unless you also click 'Stop' to stop recording below.

This tool will only generate captions for what you say. It will not generate captions for student questions and discussions unless they are picked up by your microphone (not recommended).
